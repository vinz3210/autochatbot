AutoChatbot

Auto-respond to Twitch chat commands

📡 Monitor multiple channels at once
⚡ Set custom trigger phrases & responses
🔢 Optional auto-incrementing counters
🔒 Restrict triggers to specific users

AutoChatbot monitors Twitch chat channels in real-time and automatically sends responses when specific trigger phrases are detected.

Getting Started

  • Log in with your Twitch account to authorize chat reading & sending
  • Add channels you want to monitor (e.g., your own channel)
  • Create triggers with a phrase and a response
  • Keep the page open — it runs as long as the tab is active

Trigger Phrases

Set any text as a trigger (e.g., !sun). When someone types that exact phrase in chat, AutoChatbot sends your configured response. Matching is case-insensitive.

Allowed Users

By default, anyone can activate a trigger. To restrict it, enter comma-separated usernames in the Allowed Users field. Only those users will be able to trigger it — useful for preventing spam.

Counter & User Placeholders

Use special placeholders in your response text:

  • {count} — Replaced with the current counter value (if enabled).
  • {username} — Replaced with the name of the user who triggered the command.

Example: response Hello {username}, you are visitor #{count}!

Storage

All channels, triggers, and counter values are saved in your browser's local storage. They persist across page refreshes and browser restarts.
